# Fake Review Detection in Education Sector

This project uses Machine Learning and NLP techniques to detect deceptive reviews in educational institutions.

## Features
- Fake review detection using Logistic Regression
- NLP sentiment analysis using VADER
- Power BI dashboard for insights

## Dataset
- Fake review datasets from Kaggle, Yelp, HuggingFace
- Scraped education reviews dataset

## Results
- Total Reviews: 356
- Fake Reviews Detected: 19 (~5%)

## Tools Used
- Python, Scikit-learn, NLTK
- Power BI
